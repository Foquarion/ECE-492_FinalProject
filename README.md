# ECE-492: Final Project
Repository for code for Tuffy Wolf Head PCB final project

# Hardware Notes:
Physically CUT LEG 2 of each push-button switch for proper functionality
  - SW1: Top Left
  - SW2: Bottom Right
  - Reset: Bottom Right

# Programming Notes:
Project Built using CCS (Code Composer Studio 12.8.1)

**Easy Method:**
  1. Download Repository
  2. Create New Project Workspace in CCS
  3. Select Import
  4. Select Search Directory where Repository was downloaded
  5. Check Box for "COPY PROJECTS INTO WORKSPACE"
  6. Build -> Debug -> BLINK!
